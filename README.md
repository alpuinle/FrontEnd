## Run Locally

Clone the project

```bash
  git clone https://github.com/alpuinle/FrontEnd.git
```

Go to the project directory

```bash
  cd FrontEnd
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
